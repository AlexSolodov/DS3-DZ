Файлы с данными должны лежать в той же папке, откуда запускается ноутбук

Для ориентира, в названии каждого ноутбука есть указание, по какому параметру выполнялось агрегирование данных
Например: Линейная комбинация. Прогноз размещения в месяц на площадке (регионы + уровни субъекта ) - значит агрегация 
выполнялась совместно по региону и уровням субъекта, который представляли заказчики.
Ноутбуки для небольших агрегаций (регионы, отрасли) выполняются относительно недолго (час - два, в зависимости от компьютера). 
Основную часть времени занимает подбор параметров. Ноутбук по регионам и отраслям может выполняться долго
